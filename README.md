# Random-User-API-Performance-Test_by_Mahmudul_Mahadi

## Project Summary:
In this project I have done Load and Stress Testing using jmeter

## Technology used:
- jmeter
- jdk 11 

## How to run:
- ``` git clone ```
- ``` save jmax file in bin folder of jmeter ```
- ``` run the jmax file using jmeter ```
